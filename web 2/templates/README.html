1. Order Confirmation Email

File: order-confirmation-email.html

Path: /templates/order-confirmation-email.html

Sent immediately after customer completes checkout. Includes order summary and delivery information.

Available Variables:

Variable Description Example
{{CUSTOMER_NAME}} Customer's full name John Doe
{{CUSTOMER_EMAIL}} Customer's email address john@example.com
{{CUSTOMER_PHONE}} Customer's phone number 083 402 1234
{{CUSTOMER_ADDRESS}} Delivery street address 123 Main Street
{{CUSTOMER_CITY}} Delivery city Johannesburg
{{CUSTOMER_POSTAL}} Delivery postal code 2000
{{ORDER_NUMBER}} Unique order identifier AV1704730567890
{{ORDER_DATE}} Order creation date January 8, 2026
{{DELIVERY_DATE}} Expected delivery date January 22, 2026
{{ORDER_ITEMS}} HTML table rows of ordered items <tr>...</tr>

How to Use These Templates

âœ“ Important: All variable names are wrapped in double curly braces: {{VARIABLE_NAME}}. Replace these placeholders with actual data before sending or displaying the template.

Step 1: Load Template

// Load template HTML const templateHTML = document.getElementById('order-confirmation-email').innerHTML;

Step 2: Replace Variables

function renderTemplate(template, data) { return template .replace(/{{CUSTOMER_NAME}}/g, data.customerName) .replace(/{{ORDER_NUMBER}}/g, data.orderNumber) .replace(/{{TOTAL}}/g, data.total) // ... replace all variables }

Step 3: Send via EmailJS

const renderedHTML = renderTemplate(templateHTML, orderData); emailjs.send('SERVICE_ID', 'TEMPLATE_ID', { to_email: customerEmail, email_html: renderedHTML, // ... other parameters });

Step 4: For Print/PDF

// Render template to DOM document.getElementById('invoice-container').innerHTML = renderedHTML; // Print or convert to PDF window.print(); // or use PDF library like jsPDF
